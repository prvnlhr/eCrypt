# eCrypt

## What is eCrypt ??
eCrypt is a *web application* which stores *banks card details*, *LoginIds and passwords* and *Documents* all at one place . User can access them on any device and it supports full *CRUD* operations. It is fast and fully secured and can be accessed on the go.


## Technologies used :
eCrypt is a **MERN** stack web application. It uses **Redux** for state management and **JSON Web Tokens** for authentication and authorization.
Frontend part uses Material UI for styling  



### Following are features of eCrypt::
1. User can do CRUD(Create, read, update and delete) operations on every stored item.
2. It supports dynamic search option to display results.
3. Favourites section lets you mark  items as favourite.
4. Dashboard shows all the recent activities inside the app.
5. Strong authentication and authorization.
6. Email verification for new users.
7. Forget password option.
8. Profile update section lets you update and deleted your account permanently.
9. and many more....give it a try
